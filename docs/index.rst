Introduction
-----------
Flask Admin Dashboard is a powerful administrative interface for document management with multi-tenant support.
It provides comprehensive document handling capabilities including file uploads, categorization, and user management.

Features
--------
* Multi-tenant support with company-specific data isolation
* Document management with metadata
* User authentication and authorization
* Department and category organization
* RESTful API endpoints
* Responsive user interface
